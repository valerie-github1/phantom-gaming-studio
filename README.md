## Vision

Phantom Gaming Studio builds immersive, AI-native gaming experiences — from holographic dashboards to procedurally generated open worlds — powered by the Phoenix AI platform.

We sit at the intersection of **cinematic design**, **real-time 3D**, and **large language models**, creating games that adapt, respond, and evolve with every player.

## Mission

To become the leading AI-native game studio in Europe by 2028, shipping titles that use generative AI not as a gimmick, but as a core gameplay mechanic.
